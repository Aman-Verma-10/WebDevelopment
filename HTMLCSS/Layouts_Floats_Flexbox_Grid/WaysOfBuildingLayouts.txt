What does "Layout" means
1. Layout is the way text, images and other content is placed and arranged on a webpage
2. Layout gives the page a visual structure, into which we place out content
3. Biuliding a layout: arranging page elements into visual structure, instead of simply having them placed one after another (normal flow)

Page layout vs Component layout
page layout is laying out the elements so the big pieces of content inside oa a webpage, these bigger page layouts are themelves made of components and these Components themselves need some kind of layout, these component themselves are made out of smaller pieces of content which also need to be arrange in some kind of way. i.e. Component layout

3 ways of building layouts with CSS

1. 1st technique of building layout is "float" CSS property: The old way of building layouts of all sizes, using the float CSS property. Still used, but getting outdated fast.
2. Using Flexbox: Modern way of laying out elements in a 1-dimensional row without using floats. Perfect for component layouts.
3. CSS Grid: for laying out elements in a fully-fledged 2-dimensional grid. Perfect for page layouts and complex components.
